This is my work for weeks 1-3 on the html and css worksheets. We looked at how to create basic html pages and css styles. This came very naturally to me due to my knowledge of coding. We learnt about adding images and text in various ways such as using lists, headers and footers. I could then style the text by changing its properties such as the colour, font size and weight. Following the pizza website we made one about about cake with a link for it placed at the top of the page. This helped me understand how to arrange content on my website using padding, margin and flexbox.
